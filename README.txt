interaction_areas
=================

A minetest mod to manage interaction areas.

Version: 1.0
License: GPL 3 or later

Dependencies: none

Report bugs or request help on the forum topic.

Installation
------------

Extract the archive and place the "interaction_areas" folder to the "mods" directory in your Minetest folder.

For further information or help see:
http://dev.minetest.net/Installing_Mods
